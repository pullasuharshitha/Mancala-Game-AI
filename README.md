# Mancala AI Game – Minimax vs Alpha-Beta Pruning

An AI-powered implementation of the classic **Mancala board game** where a human player competes against an intelligent AI agent.  
The AI uses **Minimax search** and **Alpha-Beta pruning** to determine optimal moves and improve search efficiency.

## Project Overview

This project demonstrates how **game tree search algorithms** can be applied to strategic board games.  
The AI evaluates possible future game states using **Minimax** and optimizes the search process using **Alpha-Beta pruning**, significantly reducing the number of explored nodes.

The project also includes a **graphical user interface** for gameplay and a **performance comparison graph** between the two algorithms.

## Features

- Human vs AI Mancala gameplay
- AI decision making using **Minimax Algorithm**
- Optimization using **Alpha-Beta Pruning**
- Node exploration comparison between algorithms
- Interactive graphical interface
- Visualization of algorithm performance

## Technologies Used
